MongoDB的php拓展

Jul 24, 2019
1 min read

MongoDB拓展下载地址：http://pecl.php.net/package/mongodb

  1. 选择对应PHP的版本下载
  2. 将php_mongodb.dll文件添加到对应版本PHP的ext文件中
  3. 修改php.ini，添加extension=php_mongodb.dll
  4. 重启Apache，查看phpinfo中是否开启MongoDB拓展

PHP操作MongoDB

开发文档：http://php.net/manual/zh/book.mongodb.php

连接MongoDB：$mongo = new MongoDB\Driver\Manager(‘mongodb://用户名:密码@主机ip:端口号/数据库名’);

<?php
$mongo = new MongoDB\Driver\Manager('mongodb://root:123456@127.0.0.1:27017/gzd');
var_dump($mongo);
?>
